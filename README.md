



# 🏟️ Play4G – Frontend

A modern sports facility booking frontend built with Next.js.

---

## 🚀 Purpose

This frontend allows users to:
- Browse sports facilities
- Search and filter facilities
- View detailed facility pages
- Book facilities after login
- View and manage bookings

---

## 🌐 Live URL

https://assignment-09-play4g.vercel.app

---

## ✨ Features

- Authentication (login/register)
- Facility listing page (dynamic from API)
- Facility search & filter
- Facility details page
- Booking system UI
- User bookings page
- Protected routes
- Responsive UI (mobile + desktop)

---

## 📦 NPM Packages Used

- next
- react-icons
- react-toastify
- better-auth
- tailwindcss
- daisyui

---

## 🔐 Auth System

- JWT-based authentication
- Token fetched via Better Auth client
- Protected API calls using Authorization headers

---

## 🛠️ Tech Stack

- Next.js (App Router)
- Tailwind CSS + DaisyUI
-  API integration
